# 🌟 About the Repository

**Python Mini Projects** is a collection of beginner-to-intermediate level Python projects designed to improve programming fundamentals through hands-on practice.

The repository contains both fun CLI games and useful utility applications that demonstrate real-world Python concepts like:
- File handling
- Randomization
- Input validation
- Logic building
- Data management
- Password generation
- Dictionaries and loops

| Dimension | Detail |
|---|---|
| 🎯 Purpose | Learning Python through projects |
| 🧱 Architecture | Independent standalone scripts |
| 💻 Project Types | Games + Utility Applications |
| 👥 Audience | Beginners & Python learners |

- Clear button functionality
- Responsive calculator buttons
- Real-time expression display
- Keyboard-friendly layout

- Python
- Tkinter

### 📚 Concepts Practiced

- Event handling
- Functions
- Button widgets
- Frames & layouts
